## PRIMARY COLOR USAGE

**Deep Navy (`#0f172a`):**
- Should dominate main page backgrounds, hero sections, and primary containers
- Used for the base foundation of the website

**Primary Blue (`#2563eb`):**
- Used for primary buttons, borders, icons, and interactive elements
- Main brand color for CTAs and key actions

**Accent Blue (`#3b82f6`):**
- Used for highlights, text accents, badges, and secondary CTAs
- Digital interaction states and emphasis elements

**Hover Blue (`#60a5fa`):**
- Used for hover states on buttons, borders, and interactive elements
- Provides visual feedback on user interactions

---

## SECONDARY COLOR USAGE

**Slate Blue (`#1e293b`):**
- Supports secondary backgrounds, cards, and section containers
- Used for elevated UI surfaces and content blocks

**Light Gray (`#cbd5e1`):**
- Used for body text, descriptions, and secondary content
- Provides readability on dark backgrounds

**Pure White (`#ffffff`):**
- Main text color for headings and important content
- Used for negative space and contrast

---

## COLOR BALANCING

- **60%** Deep Navy (`#0f172a`) - Primary backgrounds
- **20%** Slate Blue (`#1e293b`) - Secondary surfaces and cards
- **10%** Light Gray (`#cbd5e1`) - Body text and secondary content
- **5%** Primary Blue (`#2563eb`) - Buttons, borders, and primary elements
- **3%** Accent Blue (`#3b82f6`) - Highlights and accents
- **2%** Hover Blue (`#60a5fa`) - Interactive hover states
- **<1%** Pure White (`#ffffff`) - Headings and emphasis

---

## Color Relationships

### Primary Palette
- **Deep Navy** → Base foundation
- **Primary Blue** → Primary actions
- **Accent Blue** → Highlights and emphasis
- **Hover Blue** → Interactive feedback

### Supporting Palette
- **Slate Blue** → Elevated surfaces
- **Light Gray** → Readable text
- **Pure White** → Maximum contrast

---

## Usage Guidelines

1. **Backgrounds**: Use Deep Navy for main backgrounds, Slate Blue for cards/sections
2. **Buttons**: Primary Blue default, Hover Blue on hover
3. **Text**: White for headings, Light Gray for body text
4. **Borders**: Primary Blue with Hover Blue on hover
5. **Accents**: Accent Blue for badges, highlights, and secondary elements
6. **No Gradients**: All colors are solid - no gradients or glows
