# Logo Color Variations

This document describes the different color variations of the Pizza Anytime logo for client review.

## Variation 1: Monochrome Blue
**File:** `logo-variation-1-monochrome-blue.svg`
**Description:** All elements use Primary Blue (#2563eb) - clean, professional, and consistent
**Best for:** Standard use, formal presentations, print materials
**Color Scheme:** 100% Primary Blue (#2563eb)

---

## Variation 2: Primary Blue with Accent
**File:** `logo-variation-2-primary-accent.svg` (Current/Default)
**Description:** Primary Blue (#2563eb) with one slice in Accent Blue (#3b82f6) for visual depth
**Best for:** Digital use, website headers, modern branding
**Color Scheme:** 
- Main elements: Primary Blue (#2563eb)
- Accent slice: Accent Blue (#3b82f6)

---

## Variation 3: Accent Blue with Hover Accent
**File:** `logo-variation-3-accent-hover.svg`
**Description:** Lighter, more vibrant look using Accent Blue (#3b82f6) with Hover Blue (#60a5fa) accents
**Best for:** Bright backgrounds, casual settings, youth-oriented marketing
**Color Scheme:**
- Main elements: Accent Blue (#3b82f6)
- Accent slice: Hover Blue (#60a5fa)

---

## Color Reference

| Color Name | HEX Code | Usage |
|------------|----------|-------|
| Deep Navy | `#0f172a` | Dark, premium backgrounds |
| Slate Blue | `#1e293b` | Professional, subtle |
| Primary Blue | `#2563eb` | Main brand color |
| Accent Blue | `#3b82f6` | Highlights and accents |
| Hover Blue | `#60a5fa` | Light, friendly |

---

## Recommendations

- **For Website Header:** Variation 2 (Primary with Accent) - matches current site design
- **For Print Materials:** Variation 1 (Monochrome) - clean and professional
- **For Dark Backgrounds:** Variation 6 (Navy with Primary) - high contrast
- **For Light Backgrounds:** Variation 5 (Hover with Accent) - soft and approachable
- **For Corporate Use:** Variation 4 (Slate with Primary) - sophisticated
- **For Youth Marketing:** Variation 3 (Accent with Hover) - vibrant and energetic

---

## How to Use

All variations are SVG files located in `/src/assets/` directory. They can be:
- Imported directly in React components
- Used in HTML with `<img>` tags
- Converted to PNG/JPG for print materials
- Scaled to any size without quality loss
